# Disha Homeopathy Clinic Website

Frontend website for Disha Homeopathy Clinic, built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui components.

## Prerequisites

- Node.js 18 or later
- npm

## Install Dependencies

Open a terminal in the project root and run:

```bash
npm install
```

If PowerShell blocks `npm` with an execution policy error on Windows, use:

```bash
npm.cmd install
```

## Run the Project Locally

Start the development server:

```bash
npm run dev
```

Windows PowerShell fallback:

```bash
npm.cmd run dev
```

Vite will print a local URL in the terminal. In this project, the app also works at:

```text
http://127.0.0.1:5173
```

## Build for Production

Create a production build:

```bash
npm run build
```

Windows PowerShell fallback:

```bash
npm.cmd run build
```

## Preview the Production Build

After building, preview the production output locally:

```bash
npm run preview
```

Windows PowerShell fallback:

```bash
npm.cmd run preview
```

## Run Tests

Run the test suite once:

```bash
npm run test
```

Windows PowerShell fallback:

```bash
npm.cmd run test
```

Run tests in watch mode:

```bash
npm run test:watch
```

Windows PowerShell fallback:

```bash
npm.cmd run test:watch
```

## Project Scripts

- `npm run dev` - start the Vite development server
- `npm run build` - create a production build
- `npm run build:dev` - create a development-mode build
- `npm run preview` - preview the built app
- `npm run lint` - run ESLint
- `npm run test` - run Vitest once
- `npm run test:watch` - run Vitest in watch mode
